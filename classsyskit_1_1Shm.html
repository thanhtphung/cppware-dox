syskit::Shm::Shm ( size_t  size)

Construct a shared memory segment with the given size in bytes. Given size must be non-zero and must not exceed the platform-specific limit. The construction can fail due to lack of system resources or due to an unsupported size. Use isOk() to determine if the shared memory segment construction is successful.

void syskit::Shm::detach ( )

Detach from the system resource and allow it to persist. That is, after detaching, the Shm instance is no longer valid, but the system resource persits even after the Shm instance is destructed.
