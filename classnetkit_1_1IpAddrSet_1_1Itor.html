Detailed Description

set iterator

A class representing an IpAddrSet iterator. It provides a scheme to traverse the addresses in an IpAddrSet instance. Example:

IpAddrSet::Itor it(set);
unsigned int loAddr;
unsigned int hiAddr;
while (it.nextRange(loAddr, hiAddr))
{
//do something with each range in the set
}
