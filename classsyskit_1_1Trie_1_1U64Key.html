Detailed Description

unsigned 64-bit key

A class representing an unsigned 64-bit key. It helps form normalized keys used by a digit trie. This implementation breaks a 32-bit number into a sequence of 8 bytes with each byte holding 8 bits. For example, the normalized form of 0x123456789abcdef0ULL is the 08-12-34-56-78-9a-bc-de-f0 hexadecimal byte sequence. Construct an instance or use encode() to normalize a native key. Use decode() to decode a normalized key. Example:

Trie::U64Key key(k);
assert(Trie::U64Key::decode(key) == k);
