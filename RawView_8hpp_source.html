18  //! A class representing binary data with an ASCII view. The ASCII view
19  //! consists of three parts: optional hexadecimal offsets, hexadecimal
20  //! contents, and optional ASCII contents. Trivial methods are available
21  //! to configure the ASCII view (e.g., bytesPerRow(), rowsPerPage(),
22  //! showAscii(), showOffset(), spacing(), etc.). Sample default ASCII
23  //! view:
24  //!\code
25  //! [00000073] 2f2f2120 4120636c 61737320 72657072 //! A class repr
26  //! [00000083] 6573656e 74696e67 2062696e 61727920 esenting binary
27  //! [00000093] 64617461 20776974 6820616e 20415343 data with an ASC
28  //! [000000a3] 49492076 6965772e 20546865 20415343 II view. The ASC
