String appkit::D64::Vec::toString ( char  wrapStringIn = '|') const

Default format with vertical bars wrapping the vector: "|v0 v1 v2|". Example: "|6789 -0.1 22.345|". Use zero to omit the wrapping. Some wrap pairing is automatic. For example, either '[' or ']' will result in "[v0 v1 v2]".
