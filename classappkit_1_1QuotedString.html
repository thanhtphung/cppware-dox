appkit::QuotedString::QuotedString ( const String &  str,
char  quote = '\"',
bool  escapeSingleQuotedString = true 
)

Enquote given string inside given quotes. If a character has a pre-defined simple escape sequence, then its escape sequence will be used. For example: <abc'"123> --> <"abc\'"123">. The quote value is typically a single quote or a double quote. If it's zero, no surrounding quotes will be added, but special characters will still be replaced with their known escape sequences. To skip escape sequence usage in a single quoted string, set escapeSingleQuotedString to false.
