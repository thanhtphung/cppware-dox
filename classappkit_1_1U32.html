Detailed Description

unsigned 32-bit number

A class representing a 32-bit unsigned number. It also serves as a namespace for miscellaneous utilities using unsigned int as key arguments. Comparison function naming hints:

K: compare keys
KR: compare keys, reverse the sense of comparison
P: compare keys given pointers
PR: compare keys given pointers, reverse the sense of comparison
