Detailed Description

comma-separated-value row

A class representing a CSV (comma-separated-value) row. A row consists of one or more columns separated by a delimiter. By default, the column delimiter is the comma (','). A different delimiter can be specified at construction. To iterate, use the next() and/or apply() methods. Example:

String s("1,22,333");
CsvRow row(s);
:
String col;
while (row.next(col))
{
//do something with each column
}

size_t appkit::CsvRow::countCols ( ) const

Return the total number of columns in the row. Return zero if the attached row is empty. A column consists of consecutive characters terminated by the column delimiter. The last column can but does not have to be delimited by the column delimiter. That is, non-empty row consists of at least one column even if the row has no column delimiters.

Remove surrounding quotes from given column. Also resolve escape sequences treating the column as a C++ string literal. As an exception, replace two consecutive quotes, if any, with one. No-op if given column is not quoted. Return true if column was modified (due to surrounding quote removal and/or escape sequence resolution). For a single quoted column, escape sequence resolution can be skipped if unescapeSingleQuotedString is false.
