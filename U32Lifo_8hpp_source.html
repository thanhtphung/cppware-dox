14 //! last-in-first-out queue of 32-bit numbers (aka stack)
15 class U32Lifo: private U32Fifo
16  //!
17  //! A class representing a LIFO queue of 32-bit numbers. A LIFO queue is
18  //! more commonly known as a stack. The queue has an initial capacity
19  //! of capacity() items. The capacity can grow as needed if the
20  //! growth factor is set to non-zero when constructed or afterwards
21  //! using setGrowth(). If the queue is growable, growth can occur when
22  //! items are pushed softly into the queue. For a LIFO queue, items are
23  //! normally added at the top using push() and removed also from the top
24  //! using pop(). In a full queue, items can also be pushed hard with
25  //! pushHard() causing the bottom item to fall out. Implemented using
26  //! U32Fifo with the LIFO's top equivalent to the FIFO's head. Example:
27  //!\code
28  //! U32Lifo q;
29  //! q.push(item0);
30  //! :
31  //! unsigned int item1;
32  //! q.pop(item1);
33  //!\endcode
34  //!
35 {
36

42  using Growable::capacity;
