Detailed Description

atomic 32-bit number

A class representing an atomic unsigned int. Implemented using intrinsics where applicable. This is usually the most efficient way to represent numbers which must be synchronized across threads. Reference counting is an example of typical usage. As a convenience, AtomicWord is an alias of Atomic32 in x86 builds. Example:

if (refCount_++ == 0U) //refCount_ is an Atomic32 instance
{
// code to be run by the one thread which just incremented the count from zero
}
:
if (--refCount_ == 0U) //refCount_ is an Atomic32 instance
{
// code to be run by the one thread which just decremented the count to zero
}
