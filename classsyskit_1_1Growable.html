Detailed Description

growable container

An abstract base class for a container which might grow. By default, most containers do not grow. Use setGrowth() to manage growth. Use resize() to adjust the container's capacity on demand. Example:

// If growth might have occurred, use initial capacity if possible.
if (growable.canGrow())
{
unsigned int initialCap = growable.initialCap();
growable.resize(initialCap);
}
