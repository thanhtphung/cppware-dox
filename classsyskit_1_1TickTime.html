Detailed Description

time stamp in ticks

A class representing a time stamp in ticks. These ticks are not synchronized across CPU boundaries. Tick frequency is also assumed constant. As a result, the class must be used with care when ticks need to be synchronized across CPU boundaries or when tick frequency varies due to CPU idling.
