Detailed Description

unsigned 64-bit number

A class representing a 64-bit unsigned number. It also serves as a namespace for miscellaneous utilities using unsigned long long as key arguments. Comparison function naming hints:

K: compare keys
KR: compare keys, reverse the sense of comparison
P: compare keys given pointers
PR: compare keys given pointers, reverse the sense of comparison
