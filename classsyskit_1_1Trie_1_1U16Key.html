Detailed Description

unsigned 16-bit key

A class representing an unsigned 16-bit key. It helps form normalized keys used by a digit trie. This implementation breaks a 16-bit number into a sequence of 4 bytes with each byte holding 4 bits. For example, the normalized form of 0x1234U is the 04-01-02-03-04 hexadecimal byte sequence. Construct an instance or use encode() to normalize a native key. Use decode() to decode a normalized key. Example:

Trie::U16Key key(k);
assert(Trie::U16Key::decode(key) == k);
