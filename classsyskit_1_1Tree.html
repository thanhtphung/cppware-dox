Detailed Description

2-3-4 tree of opaque items

A class representing a 2-3-4 tree of opaque items. A 2-3-4 tree is a balanced tree with nodes having 2 or 3 or 4 links. A 2-link node holds 1 item. A 3-link node holds 2 items. A 4-link node holds 3 items. For comparison, a binary tree is a tree with 2-link nodes, and a red-black tree is a binary tree representation of a 2-3-4 tree.
