Detailed Description

null-terminated string key

A class representing a null-terminated string key. It helps form normalized keys used by a digit trie. For example, the normalized form of the ASCII "abc123" string is the 06-61-62-63-31-32-33 hexadecimal byte sequence. The normalized form of the hexadecimal "abc123" string is the 06-0a-0b-0c-01-02-03 hexadecimal byte sequence. The normalized form of the decimal "abc123" string is unpredictable since it contains non-decimal characters.
