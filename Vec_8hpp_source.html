18 //! vector of opaque items
19 class Vec: public Growable
20  //!
21  //! A class representing an unsorted vector of opaque items. The vector
22  //! has an initial capacity of capacity() items. The capacity can grow
23  //! as needed if the growth factor is set to non-zero when constructed or
24  //! afterwards using setGrowth(). If the vector is growable, growth can
25  //! occur when items are added. Items are added using various forms of
26  //! the add() method and removed using various forms of the rm() method.
27  //! Updates can be expensive if many items need to be shifted up or down.
28  //! For best update performance, items should be added to the tail end,
29  //! and items should be removed without maintaining the current ordering.
30  //! Example:
31  //!\code
32  //! Vec vec;
33  //! vec.add(item0);
34  //! vec.addAtIndex(1 /*index*/, item1, false /*maintainOrder*/);
35  //! vec.addAtIndex(0 /*index*/, item2, true /*maintainOrder*/);
36  //! vec.rmFromIndex(1 /*index*/, false /*maintainOrder*/);
37  //! vec.rmFromIndex(0 /*index*/, true /*maintainOrder*/);
38  //!\endcode
39  //!
