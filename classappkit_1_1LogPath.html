Detailed Description

circular log path

A class representing a path to a primitive circular log. The circular log resides in the application's log directory. The member log files are named using this suffix pattern: "<name><index>.txt". The name and index parts can be specified during construction. As an example, LogPath("xxx-",'0','2') is a three-member circular log with these suffixes: "xxx-0.txt", "xxx-1.txt", and "xxx-2.txt".
