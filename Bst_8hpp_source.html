16 //! binary search table of opaque items
17 class Bst: public Growable
18  //!
19  //! A class representing a binary search table of opaque items. The table has
20  //! an initial capacity of capacity() items. The capacity can grow as needed if
21  //! the growth factor is set to non-zero when constructed or afterwards using
22  //! setGrowth(). If the table is growable, growth can occur when items are added.
23  //! Items are added using the add() method and removed using various forms of the
24  //! rm() method. Searches are provided by the find() methods. The binary search
25  //! methods are efficient in general, but any method requiring updates might have
26  //! to shift many items up or down. If updates are frequent and the table capacity
27  //! is relatively large, consider using some Vec class if items don't need to be
28  //! in order at all times, or some Tree/HashTable class if items do need to be in
29  //! order. Example:
30  //!\code
31  //! Bst bst;
32  //! :
33  //! bst.add(item0); //add item0 to table
34  //! bst.rm(item1); //remove item1 from table
35  //! :
36  //! bst = vec; //populate table with a Vec instance
37  //!\endcode
38  //!
