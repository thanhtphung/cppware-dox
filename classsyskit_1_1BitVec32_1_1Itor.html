Detailed Description

bit vector iterator

A class representing a BitVec32 iterator. It provides a scheme to traverse the bits in a BitVec32 instance. Example:

BitVec32::Itor it(vec);
it.applyToSetBits(cb); //apply cb() to each set bit in vec

Iterate the bit vector from low to high if loToHi is true. Iterate the bit vector from high to low otherwise. Invoke callback at each clear bit. bool syskit::BitVec32::Itor::nextClearBit ( size_t &  bit)

Determine the next clear bit. Return true if there's one. Return false otherwise (if remaining bits are all set or if there's no more bits to iterate). Behavior is unpredictable if the iterator is unattached. The first invocation after construction or reset() will find the first clear bit.
