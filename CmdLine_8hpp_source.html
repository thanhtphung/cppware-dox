17 class CmdLine
18  //!
19  //! A class representing a command line. A command line consists of a
20  //! command name, optional options, and optional arguments. An option
21  //! starts with a single hyphen ('-') or a double hyphen ("--"), and an
22  //! argument does not. Options must be specified before arguments are
23  //! given. If options are duplicated, only the last one is significant.
24  //! For example, in the "cmd -n=1 -n=2" command line, the "-n=1" option
25  //! would be ignored. This implementation uses a vector to hold arguments
26  //! and a map to hold options. This class supports the following syntax:
27  //!\code
28  //! cmdName [-opt[=opt]]* [arg...]
29  //! cmdName [--opt[=opt]]* [arg...]
30  //!\endcode
31  //! Use instance() to access the singleton command line for the calling
32  //! process. Use arg() to access the command line arguments. arg(0) is
33  //! the command name. arg(1) is the first argument. Use opt() to access
34  //! the options. opt() returns zero if the requested option is not in
35  //! the command line arguments. opt() returns an empty string for
36  //! options which have no values. Parsing of options is done in a case
37  //! insensitive manner. Helper classes are provided to ease parameter value
38  //! extraction. Refer to classes like U32 and U64 for more details. Example:
39  //!\code
40  //! const CmdLine& cmdLine = CmdLine::instance();
41  //! if (cmdLine.numArgs() == 1) //no arguments?
42  //! {
43  //! :
44  //! }
45  //!\endcode
46  //!
