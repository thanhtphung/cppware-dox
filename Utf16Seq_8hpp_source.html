17  //!
18  //! A class representing a sequence of UTF16 characters. UTF16 characters
19  //! are most efficiently accessed as an array of 32-bit values. However,
20  //! the doubled memory usage is questionable if the majority of characters
21  //! require just 16 bits each. This class can help reduce memory usage and
22  //! can also help ease interoperability. Use convertX() to convert from
23  //! other UTF forms. This class uses the UTF16 definition as described in
24  //! the Utf16 class. Example:
25  //!\code
26  //! Utf16Seq seq;
27  //! utf8_t raw[3] = {0xaaU, 0xbbU, 0xccU};
28  //! if (s.convert8(raw, 3))
29  //! {
30  //! unsigned int numChars = seq.numChars();
31  //! utf32_t* s = seq.expand();
32  //! :
33  //! // access UTF16 characters as an array of 32-bit values
