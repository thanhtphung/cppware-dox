bool netkit::Paddr::reset ( const char *  s,
char  delim = ':',
bool  portIsOptional = false 
)

Reset instance with given port address (e.g., "172.16.16.1:8080"). If portIsOptional is set, then assume port is zero (e.g., "172.16.16.1" would be the same as "172.16.16.1:0"). If delim is non-zero, given port address must use that delimiter to delimit the address and the port parts. If delim is zero, any delimiter is acceptable. Return true if s specifies a valid port address. Use of a hostname instead of the dotted notation is allowed. See IpAddr for details.
