Detailed Description

unsigned 32-bit key

A class representing an unsigned 32-bit key. It helps form normalized keys used by a digit trie. This implementation breaks a 32-bit number into a sequence of 8 bytes with each byte holding 4 bits. For example, the normalized form of 0x12345678UL is the 08-01-02-03-04-05-06-07-08 hexadecimal byte sequence. Construct an instance or use encode() to normalize a native key. Use decode() to decode a normalized key. Example:

Trie::U32Key key(k);
assert(Trie::U32Key::decode(key) == k);
