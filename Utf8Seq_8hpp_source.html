20  //! A class representing a sequence of UTF8 characters. UTF8 characters
21  //! are most efficiently accessed as an array of 32-bit values. However,
22  //! the quadrupled memory usage usually cannot be justified if the majority

24  //! can also help ease interoperability. Use convertX() to convert
25  //! from other UTF forms. This class uses the UTF8 definition as described
26  //! in the Utf8 class. Example:

29  //! utf16_t raw[3] = {0xaaaaU, 0xbbbbU, 0xccccU};
30  //! if (seq.convert16(raw, 3, Utf8Seq::DEFAULT_CHAR))
31  //! {
32  //! unsigned int numChars = seq.numChars();
33  //! utf32_t* s = seq.expand();
34  //! :
