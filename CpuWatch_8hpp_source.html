12 //! stop watch with CPU time
13 class CpuWatch
14  //!
15  //! A class representing a stop watch with CPU time. It holds the
16  //! start time when the watch was started, the stop time when the
17  //! watch was stopped, and the stop-start time difference. In addition,
18  //! it holds the CPU time used by the calling process in the stop-start
19  //! duration. The system CPU time is the time spent in kernel mode. The
20  //! user CPU time is the time spent in user mode. The time stamps are
21  //! 64-bit tick counts. The tick resolution (i.e., ticks per second)
22  //! is platform dependent. Use TickTime to interpret the tick counts.
23  //! On windows, CPU time used by the calling thread (instead of the
24  //! calling process) can be obtained using the useThreadTime parameter
25  //! during construction and/or reset. The parameter has no significance
26  //! on other platforms.
27  //!
28 {
