15 class U64Vec: public Growable
16  //!
17  //! A class representing an unsorted vector of 64-bit numbers. The vector
18  //! has an initial capacity of capacity() items. The capacity can grow
19  //! as needed if the growth factor is set to non-zero when constructed or
20  //! afterwards using setGrowth(). If the vector is growable, growth can
21  //! occur when items are added. Items are added using various forms of
22  //! the add() method and removed using various forms of the rm() method.
23  //! Updates can be expensive if many items need to be shifted up or down.
24  //! For best update performance, items should be added to the tail end,
25  //! and items should be removed without maintaining the current ordering.
26  //! Example:
27  //!\code
28  //! U64Vec vec;
29  //! vec.add(item0);
30  //! vec.addAtIndex(1 /*index*/, item1, false /*maintainOrder*/);
31  //! vec.addAtIndex(0 /*index*/, item2, true /*maintainOrder*/);
32  //! vec.rmFromIndex(1 /*index*/, false /*maintainOrder*/);
33  //! vec.rmFromIndex(0 /*index*/, true /*maintainOrder*/);
34  //!\endcode
35  //!
