Detailed Description

xml document prolog

Document prolog. A prolog contains an XML declaration and zero or more miscellaneous items (comments, etc.).

Construct prolog with given declaraction and no miscellaneous items. Its XML form is an empty string if given attributes are empty. For version "1.1", encoding "utf-8", this instance would generate this XML stream: "<?xml version="1.1" encoding="utf-8"?>".
