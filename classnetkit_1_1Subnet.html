netkit::Subnet::Subnet ( const char *  id,
bool  skipNormalization = false 
)
inline

Construct instance from given address and mask. Also normalize the address part by ignoring the host bits. For example, "192.168.3.1/24" is normalized as "192.168.3.0/24". The following formats are allowed: "a.b.c.d/n", "a.b.c.d/a.b.c.d", and "a.b.c.d". For the "a.b.c.d" form, the mask is derived from its network class assuming classful subnetting. For example, "172.16.0.0" is equivalent to "172.16.0.0/16" because the address is in the B class. Use skipNormalization to skip normalizing the address part. Use isOk() to see if instance is valid.

bool  useCidrMode = true,
unsigned int  netMask = 0U 
) const

Form bit mask string. Use '0' and '1' to show leading bits in classful subnetting. Use 'n' to show network bits. Use 's' to show subnet bits. Use 'h' to show host bits. For example, "110nnnnn.nnnnnnnn.nnnnnnnn.ssshhhhh" is the bit mask string for "192.168.3.96/255.255.255.224" in classful subnetting. The same subnet would have the following bit mask string in classless subnetting: "nnnnnnnn.nnnnnnnn.nnnnnnnn.ssshhhhh".

Return the host address range for this subnet. For a 32-bit mask, the subnet address is the range.
