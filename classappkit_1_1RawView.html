Detailed Description

raw data view

A class representing binary data with an ASCII view. The ASCII view consists of three parts: optional hexadecimal offsets, hexadecimal contents, and optional ASCII contents. Trivial methods are available to configure the ASCII view (e.g., bytesPerRow(), rowsPerPage(), showAscii(), showOffset(), spacing(), etc.). Sample default ASCII view:

[00000073] 2f2f2120 4120636c 61737320 72657072 //! A class repr
[00000083] 6573656e 74696e67 2062696e 61727920 esenting binary
[00000093] 64617461 20776974 6820616e 20415343 data with an ASC
[000000a3] 49492076 6965772e 20546865 20415343 II view. The ASC
