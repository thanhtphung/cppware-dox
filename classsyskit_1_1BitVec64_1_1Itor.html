Detailed Description

bit vector iterator

A class representing a BitVec64 iterator. It provides a scheme to traverse the bits in a BitVec64 instance. Example:

BitVec64::Itor it(vec);
it.applyToSetBits(cb); //apply cb() to each set bit in vec

Iterate the bit vector from low to high if loToHi is true. Iterate the bit vector from high to low otherwise. Invoke callback at each clear bit. Determine the next clear bit. Return true if there's one. Return false otherwise (if remaining bits are all set or if there's no more bits to iterate). Behavior is unpredictable if the iterator is unattached. The first invocation after construction or reset() will find the first clear bit.
