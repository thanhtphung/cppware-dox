String appkit::D64::Quartet::toString ( char  wrapStringIn = '|') const

Default format with vertical bars wrapping the quartet: "|v0 v1 v2 v3|". Example: "|678.9 -0.1 22.345 0|". Use zero to omit the wrapping. Some wrap pairing is automatic. For example, either '<' or '>' will result in "<v0 v1 v2 v3>".
