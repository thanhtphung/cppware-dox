bool netkit::IpAddr::reset ( const char *  s)

Reset instance with given address. Return true if successful. That is, if s results in a valid dotted IP address. For example, "172.16.16.1". A hostname is valid if it can be translated to an IPv4 address. An empty string results in a registered address of the local computer, and "localhost" results in the loopback address. Instance is not updated if unsuccessful.
